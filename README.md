#  npmclass v1.0.0

The [npmclass](https://xxxx.com/) library exported as [Node.js](https://nodejs.org/) modules.

## Installation

Using npm:
```shell
$ npm i -g npm
$ npm i --save npmclass
```

In Node.js:
```js
// Load the full build.
var _ = require('npmclass');
```

See the [package source]() for more details.
